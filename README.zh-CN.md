# Bit Slice

> [English](README.md)

FPGA 位切片与对齐工具 — 基于浏览器的十六进制数据位级可视化、搜索和重新对齐工具。

**在线演示**: [sure-wang.github.io/bit-slice](https://sure-wang.github.io/bit-slice/)

## 功能

- **Hex 转二进制可视化** — 将十六进制数据转为二进制位串并展示
- **窗口切片** — 拖动滑条选择起始位，设定窗口宽度对数据进行分组
- **位序列匹配** — 支持 Verilog 风格（`N'hXX`）或原始二进制模式搜索，高亮并导航
- **切片重组** — 将二进制数据重新切片为十六进制分组，验证对齐结果
- **输入历史** — 基于 localStorage 的输入历史记录，支持防抖保存

## 使用方法

打开[在线演示](https://sure-wang.github.io/bit-slice/)或在浏览器中打开本地 `index.html`。

1. 在输入框中输入十六进制数据（如 `A3F2`）
2. 拖动滑条选择起始位，调整窗口大小
3. 可选：输入匹配模式（如 `7'h3F` 或 `0011111`）查找并跳转到目标位序列
4. 结果区域展示重新切片后的十六进制分组

## 限制

- **输入上限**: 最多 512 个十六进制字符（2048 位）
- **窗口大小**: 每个切片最多 2048 位 — `binaryToHex` 使用 BigInt，在此范围内无精度限制
